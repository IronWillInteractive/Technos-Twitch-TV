# PRO-CRAWL OS TelepromptJS Professional Local

Run `LaunchTool.BAT` on Windows, or run `python server.py` from this folder and open the printed localhost URL.

## Pages

- `index.html` - launch console with loader
- `Tools/TelePROmpt.html` - desktop teleprompter studio
- `Mobile/TelePROmptMobile.html` - mobile/touch teleprompter entry
- `Docs/Readme.html` - local documentation

## Storage

Custom presets are saved in browser localStorage. No backend, account, or cloud database required.
