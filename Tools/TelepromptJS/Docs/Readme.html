LOCAL FRONT-END TELEPROMPTER SYSTEM

PRO-CRAWL OS Docs

This package runs as a local static front-end app. It stores scripts and custom presets in the browser using localStorage, so no account, database, or cloud backend is required.

Included Files

index.html

Professional landing page with the requested loader animation and launch buttons.

Main desktop teleprompter studio with presets, editor, controls, fullscreen, guide bar, and mirror modes.

Touch-first entry using the same core code plus the mobile dock and responsive layout.

Shared professional styling plus mobile/touch media queries.

How To Run

Double click LaunchTool.BAT on Windows. It starts a local server and opens the launch console. You can also open index.html directly, but the local server is cleaner for browser security and linked files.

Preset System

Use the preset title field and SAVE button to store the current script. Built-in presets can be loaded or copied; saved presets can be edited or deleted. Data is stored under pro_crawl_user_presets_v1 in localStorage.

Mobile Notes

The mobile build uses larger touch targets, a bottom quick-control dock, safe-area viewport support, landscape handling, and touch scroll syncing. Use Mobile/TelePROmptMobile.html on phones or tablets.
